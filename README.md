# About Me:
Backend software engineer based in Lisbon, focused on distributed systems and high-performance backend infrastructure. Currently working on a low-latency FX pricing engine in a fintech environment, with prior experience building microservices for EV charging systems at BMW Group. My research background is in fault-tolerant distributed systems and Saga-based microservice architectures.

Outside of backend engineering, I like building small tools that solve real problems I run into.

🏊‍♂️🚴‍♂️🏃‍♂️ 3.8km, 180km, 42km - finished an Ironman in 2024, and I'm pretty sure my legs still haven't forgiven me.
